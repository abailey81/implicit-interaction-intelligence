## Refresh

```bash
# 1. Server must already be running (uvicorn server.app:app --port 8000)
poetry run python scripts/ui_smoke_test.py --out docs/screenshots
```

The script asserts that every tab loads without console errors and that
the `tab-panel` div is non-empty before saving the PNG. If any tab
fails, the script exits non-zero and the gallery is left unchanged.
