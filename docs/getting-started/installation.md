# Installation

This page walks you from a clean machine to a working I³ development
environment, ready to train models, run the demo server, and execute the
test suite.

!!! tip "Just want to try the demo?"
    Skip to the [Quickstart](quickstart.md) — five minutes, no training
    required.

## Prerequisites { #prerequisites }

| Requirement | Minimum | Recommended |
|:------------|:--------|:------------|
| **Python**        | 3.10          | 3.12                       |
| **Poetry**        | 1.8           | 1.8 or newer               |
| **Disk space**    | 1 GB          | 5 GB (with checkpoints)    |
| **RAM**           | 4 GB          | 8 GB                       |
| **OS**            | Linux / macOS / Windows 10+ | Linux / macOS |
| **Anthropic API key** | optional  | required for cloud routing |
| **GPU**           | not required  | CUDA 11.8+ for training    |

!!! note "Why Poetry?"
    I³ locks its dependency graph with Poetry's `poetry.lock`. See
    [ADR 0006 — Poetry over pip-tools](../adr/0006-poetry-over-pip-tools.md)
    for the rationale.

## One-shot setup { #one-shot }

=== "Linux / macOS"

    ```bash
    git clone https://github.com/abailey81/implicit-interaction-intelligence.git
    cd implicit-interaction-intelligence
    ./scripts/setup.sh
    ```

=== "Windows (PowerShell)"

    ```powershell
    git clone https://github.com/abailey81/implicit-interaction-intelligence.git
    Set-Location implicit-interaction-intelligence
    python -m venv .venv
    .\.venv\Scripts\Activate.ps1
    pip install poetry
    poetry install --with dev,security,docs
    Copy-Item .env.example .env
    python scripts\generate_encryption_key.py
    ```

## Manual setup { #manual }

### 1. Clone the repository

```bash
git clone https://github.com/abailey81/implicit-interaction-intelligence.git
cd implicit-interaction-intelligence
```

### 2. Install Python and Poetry

```bash
# Python 3.10, 3.11, or 3.12 is required.
python3 --version

# Install Poetry 1.8+
curl -sSL https://install.python-poetry.org | python3 -
```

### 3. Install project dependencies

```bash
poetry install --with dev,security,docs
```

This installs:

- **Runtime**: PyTorch ≥ 2.6, FastAPI ≥ 0.115, Pydantic ≥ 2.6,
  `cryptography` ≥ 43, `aiosqlite`, `httpx`, `numpy`, `scikit-learn`.
- **Dev**: `pytest`, `pytest-asyncio`, `ruff`, `mypy`, `pre-commit`.
- **Security**: `bandit`, `pip-audit`, `safety`.
- **Docs**: MkDocs Material + plugins (see below).

### 4. Create environment file and encryption key

```bash
cp .env.example .env
python scripts/security/generate_encryption_key.py
```

Edit `.env`:

```dotenv title=".env"
# Required if you want to exercise the cloud routing path
ANTHROPIC_API_KEY=sk-ant-...

# Generated by scripts/security/generate_encryption_key.py — DO NOT commit
I3_ENCRYPTION_KEY=<32-byte-url-safe-base64-fernet-key>

# Allow browser clients from these origins (comma separated)
I3_CORS_ORIGINS=http://localhost:8000,http://127.0.0.1:8000

# Flip on to expose /demo/reset and /demo/seed
I3_DEMO_MODE=true
```

!!! warning "Never commit `.env`"
    The `.gitignore` already excludes it. The encryption key grants access
    to every user profile at rest. Treat it like a database password.

### 5. Verify the install

```bash
poetry run pytest -q
poetry run python -c "import i3; print(i3.__version__)"
```

## Optional components { #optional }

### CUDA for training

If you have an NVIDIA GPU and CUDA 11.8 or newer:

```bash
poetry run pip install --upgrade "torch>=2.6" --index-url https://download.pytorch.org/whl/cu121
```

Training time drops from approximately two hours (laptop CPU) to under five
minutes on an RTX 4090. See [Training](training.md).

### Documentation site

```bash
poetry install --with docs
poetry run mkdocs serve
# Open http://127.0.0.1:8000
```

### Pre-commit hooks

```bash
poetry run pre-commit install
```

This enables `ruff`, `black`, and `mypy` checks on every commit.

## Verifying your install { #verifying }

Run the full acceptance suite:

```bash
make test         # 80+ unit + integration tests, ~30 s
make lint         # ruff + mypy
make security     # bandit + pip-audit
```

If every command exits with `0`, you are ready to proceed to the
[Quickstart](quickstart.md).

## Troubleshooting { #troubleshooting }

| Symptom | Likely cause | Fix |
|:--------|:-------------|:----|
| `ModuleNotFoundError: torch` | Poetry install skipped the torch wheel | `poetry lock --no-update && poetry install` |
| `RuntimeError: Fernet key must be 32 url-safe base64-encoded bytes` | Empty or malformed `I3_ENCRYPTION_KEY` | Re-run `scripts/security/generate_encryption_key.py` |
| `ImportError: cannot import name '…' from 'pydantic'` | Pydantic v1 installed | `poetry update pydantic pydantic-settings` |
| Windows `DLL load failed while importing _C` | Mixed CUDA builds | Uninstall any system-wide PyTorch; reinstall inside the Poetry venv |
| Permission errors on `data/profiles.sqlite` | Fernet key changed between runs | Delete `data/profiles.sqlite` and regenerate |

See the [full troubleshooting guide](../operations/troubleshooting.md) for more.
